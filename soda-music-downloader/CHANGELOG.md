# 更新日志

## [1.0.0] - 2025-10-13

### 新增功能
- 🎵 自动监听剪贴板中的汽水音乐分享链接
- 🎤 智能提取艺术家和歌曲名信息
- 🔄 自动转换MP4为MP3格式
- 🚫 去重功能，避免重复下载
- 📝 下载历史记录管理
- 🎯 精确过滤，只处理汽水音乐链接
- 📁 自动创建规范的文件名格式：`艺术家 - 歌曲名.mp3`

### 核心脚本
- `auto_download_final.sh` - 主要下载脚本
- `manage_history.sh` - 历史记录管理工具
- `get_share_link.sh` - 分享链接获取助手

### 技术特性
- 基于Selenium的网页解析
- 使用MoviePy进行音频格式转换
- 支持Chrome WebDriver自动管理
- 完善的错误处理和日志记录

### 系统支持
- macOS 10.15+ (Catalina或更高版本)
- Python 3.7+
- Chrome浏览器

### 文档
- 完整的README.md说明文档
- 详细的安装指南(INSTALL.md)
- 示例文件和使用说明
- MIT开源协议
