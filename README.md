# 汽水音乐下载器

一个自动监听剪贴板并下载汽水音乐的工具，支持音频下载和歌词提取。

## 预览效果

https://github.com/user-attachments/assets/preview.mov

> 演示视频展示了完整的使用流程：启动程序 → 复制链接 → 自动下载

## 功能特点

- 🎵 **自动监听剪贴板**：复制汽水音乐链接即可自动下载
- 📝 **歌词提取**：自动提取并保存LRC格式歌词文件
- 🔄 **格式转换**：自动将MP4转换为MP3格式（可选）
- 📋 **历史记录**：避免重复下载相同链接
- 🎯 **智能识别**：只处理汽水音乐链接，忽略其他内容
- 📁 **文件整理**：按"艺术家 - 歌曲名"格式命名文件

## 系统要求

- macOS系统
- Python 3.6+
- 网络连接

## 安装方法

1. 克隆或下载项目到本地
2. 进入项目目录
3. 运行安装脚本：

```bash
chmod +x install.sh
./install.sh
```

## 使用方法

1. 启动下载器：
```bash
./auto_download.sh
```

2. 复制汽水音乐分享链接到剪贴板
   - 在汽水音乐App中点击分享按钮
   - 选择"复制链接"

3. 程序会自动检测并开始下载

4. 按 `Ctrl+C` 停止程序

## 下载文件说明

- **音频文件**：`艺术家 - 歌曲名.mp3`
- **歌词文件**：`艺术家 - 歌曲名.lrc`
- **历史记录**：`download_history.txt`

## 依赖说明

- `requests`：网络请求库
- `moviepy`：音频转换库（可选，如果未安装会使用备用方案）

## 注意事项

1. 仅支持汽水音乐官方分享链接
2. 下载的内容仅供个人学习使用
3. 请遵守相关版权法律法规
4. 如果转换失败，会保留原始MP4文件

## 故障排除

### 常见问题

**Q: 提示"未找到音频URL"**
- A: 可能是链接格式不正确或网络问题，请重试

**Q: 下载的文件很小或损坏**
- A: 可能是网络不稳定，请检查网络连接后重试

**Q: 转换MP3失败**
- A: 程序会自动保留原始MP4文件，可手动转换
